# Testing Documentation

## Test Structure

### Unit Tests

- Agent functionality
- Configuration handling
- Message processing

### Integration Tests

- Multi-agent interactions
- API integration
- UI/Backend communication

### End-to-End Tests

- Complete user workflows
- System performance
- Error handling

## Running Tests

```bash
# Run all tests
pytest

# Run specific test category
pytest tests/unit
pytest tests/integration
pytest tests/e2e
```
